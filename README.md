## 📝 Modelo de Writeup

Cada writeup segue esta estrutura:

```
## 🎯 Objetivo
Descrição do que o lab propõe.

## 🧰 Ferramentas Usadas
Lista das ferramentas utilizadas.

## 🔍 Reconhecimento
Etapa de enumeração e coleta de informações.

## 💥 Exploração
Passo a passo do ataque/solução.

## 🚩 Flag / Resultado
Resultado final obtido.

## 📚 O que aprendi
Principais aprendizados do lab.
```
